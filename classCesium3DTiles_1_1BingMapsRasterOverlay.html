Detailed Description

A RasterOverlay that uses Bing Maps as the source for the imagery data.

urlThe url of the Bing Maps server hosting the imagery.

Begins asynchronous creation of the tile provider for this overlay and eventually returns it via a Future.

The created tile provider will not be returned via getTileProvider. This method is primarily useful for overlays that aggregate other overlays.

Parameters
asyncSystemThe async system used to do work in threads.
pAssetAccessorThe interface used to download assets like overlay metadata and tiles.
pCreditSystemThe CreditSystem to use when creating a per-TileProvider Credit.
pPrepareRendererResourcesThe interface used to prepare raster images for rendering.
pLoggerThe logger to which to send messages about the tile provider and tiles.
pOwnerThe overlay that owns this overlay, or nullptr if this overlay is not aggregated.
Returns
The future that contains the tile provider when it is ready, or the nullptr in case of an error.
