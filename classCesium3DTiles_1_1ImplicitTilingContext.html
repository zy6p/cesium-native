Detailed Description

A tiling context that was created for terrain tiles.

A terrain tileset is a multi-resolution quadtree pyramid of heightmaps, as described in the quantized-mesh-1.0 specification. The URLs for the individual tiles are computed from the base URL of the tileset.

std::vector<std::string> Cesium3DTiles::ImplicitTilingContext::tileTemplateUrls

The templates for the relative URLs of tiles.

The template elements of these URLs may be x, y, or z (or level), and will be substituted with the corresponding information from a CesiumGeometry::QuadtreeTileID. The version template element will be substituted with the version number of the owning context.
