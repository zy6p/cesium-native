Detailed Description

The result of applying a RasterOverlayTile to geometry.

Instances of this class are used by a Tile in order to map imagery data that is given as RasterOverlayTile instances to the 2D region that is covered by the tile geometry.

◆ MoreDetailAvailable

enum Cesium3DTiles::RasterMappedTo3DTile::MoreDetailAvailable
strong

Tile availability states.

Values of this enumeration are returned by update, which in turn is called by Tile::update. These values are used to determine whether a leaf tile has been reached, but the associated raster tiles are not yet the most detailed ones that are available.

Enumerator
No 

There are no more detailed raster tiles.

Yes

Unknown 

It is not known whether more detailed raster tiles are available.
