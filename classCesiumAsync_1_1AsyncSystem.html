Detailed Description

A system for managing asynchronous requests and tasks.

Instances of this class may be safely and efficiently stored and passed around by value.

Future<typename Impl::RemoveFuture< typename std::invoke_result<Func>::type>::type> CesiumAsync::AsyncSystem::runInMainThread ( Func &&  f) const
inline

Runs a function in the main thread, returning a promise that resolves when the function completes.

The supplied function will not be called immediately, even if this method is invoked from the main thread. Instead, it will be queued and called the next time dispatchMainThreadTasks is called.

If the function itself returns a Future, the function will not be considered complete until that returned Future also resolves.
