Future<T> CesiumAsync::Future< T >::catchInMainThread ( Func &&  f) &&
inline

Registers a continuation function to be invoked in the main thread when this Future rejects, and invalidates this Future.

The supplied function will not be invoked immediately, even if this method is called from the main thread. Instead, it will be queued and invoked the next time AsyncSystem::dispatchMainThreadTasks is called.

If the function itself returns a Future, the function will not be considered complete until that returned Future also resolves.

Any then continuations chained after this one will be invoked with the return value of the catch callback.

template<class T >
std::variant<T, std::exception> CesiumAsync::Future< T >::wait ( )
inline

Waits for the future to resolve or reject and returns the result.

This method must not be called from the main thread, the one that calls AsyncSystem::dispatchMainThreadTasks. Doing so can lead to a deadlock because the main thread tasks will never complete while this method is blocking the main thread.

Returns
The value if the future resolves successfully, or the exception if it rejects.
